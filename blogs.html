Question-1: What are the different ways to select an element in the DOM?

There are 5 ways to select an element in the DOM. These are:
1.Class selector: Using the class name the element can be selected.
example: document.getElementsByClassName('Class-Name')
2.Id selector: Using the id name the element can be selected.
example: document.getElementById('Id-Name')
3.Query Selector: Using the id and class name the element can be selected. It will give only the first element.
example: document.querySelector(".class-name or #id-name")
4.Query Selector All: Using the id and class name the element can be selected. It will give the set of all elements.
example: document.querySelectorAll(".class-name or #id-name")
5.Tag Selector: Using the tag name the element can be selected. It will give the set of all elements.
example: document.getElementsByTagName("Tag-Name")

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML: It is the property to set the element into a HTML tag. innerText: It is the property to set the text content into a HTML tag. innerText: It is the property to get and set the text content into a HTML tag.

Question-3: What is event delegation in the DOM?

Event delegation is a technique to capture an event by using the parent.For example, we have 3 items of order list. We set an event listener on the parent tag ul. When any item of the ul is clicked, we can easily find item by using the event delegation technique.

Question-4: What is event bubbling in the DOM?

Event bubble is the technique of DOM. It means when an event is occurred the event bubble up from the target element to child element.
